# 迭代 2：文件管理与 Monaco Editor

## 迭代目标
实现真实的文件操作功能，集成 Monaco Editor，支持文件内容的编辑和保存。

## 技术选型
- **编辑器**：Monaco Editor（VS Code 同款）
- **Markdown 渲染**：marked 或 react-markdown
- **文件监听**：前端轮询或 WebSocket（可选）

## 技术难点

1. **Monaco Editor 体积优化**
   - 考虑懒加载
   - 只加载 Markdown 语言支持

2. **文件树拖拽实现**
   - HTML5 Drag and Drop API
   - 或 react-beautiful-dnd

3. **自动保存机制**
   - 防抖处理避免频繁保存
   - 保存失败的重试逻辑
